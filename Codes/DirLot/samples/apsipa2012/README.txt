* Sample codes for the following reference:

- Shogo Muramatsu, Natsuki Aizawa and Masahiro Yukawa, ''Image Restoration 
  with Union of Directional Orthonormal DWTs,'' Proc. of APSIPA ASC 2012, 
  Dec. 2012.

** Requirements

- MATLAB function ssim_index.m from http://www.cns.nyu.edu/~lcv/ssim/

** Recomendation

-- Parallel Computing Toolbox

** Brief introduction

- (Optional) Open pool of MATLAB sessions for parallel computation

 >> matlabpool

- Run an M-file of which name begins with 'main' or 'disp,' 
  such as

 >> main_dirsowtimdb
  :
 >> main_sweeplambdadb
 >> main_sweeplambdasr
 >> main_sweeplambdaip
 >> disp_sweepresults
