* Sample codes for the following reference:

- Shogo Muramatsu, 
  ''SURE-LET Image Denoising with Multiple DirLOTs,''
   PCS 2012

** Requirements

- OWT_SURELET from http://bigwww.epfl.ch/demo/suredenoising/
- MATLAB function ssim_index.m from http://www.cns.nyu.edu/~lcv/ssim/

** Recomendation

-- Parallel Computing Toolbox

** Brief introduction

- (Optional) Open pool of MATLAB sessions for parallel computation

 >> matlabpool

- Run an M-file of which name begins with 'main' or 'disp,' 
  such as

 >> main_sureletshrinkage
 >> disp_denoisingresults
 >> disp_denoisedpictures
